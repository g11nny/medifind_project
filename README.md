# MediFind — Sanlı dárixana texnologiyası 💊

**MediFind** — Qaraqalpaqstanda dárilerdi tez hám ańsat tabıwǵa arnalǵan innovaciyalıq platforma.

## ✨ Imkaniyatları:
- 🔍 Dári-darmaqlardı atı boyınsha izlew.
- 📍 Eń jaqın dárixanalardı kartada kóriw.
- 💰 Dári bahaların salıstırıw.
- 🗣️ Qaraqalpaq, ózbek hám rus tillerinde kómek.

## 🛠️ Texnologiyalar:
- **Language:** Python
- **Framework:** Streamlit
- **Data:** Pandas & SQLite
- **Design:** Figma

## 🚀 Proektti iske túsiriw:
1. Kitapxanalardı ornatıń: `pip install streamlit pandas`
2. Iske túsiriń: `streamlit run main.py`
